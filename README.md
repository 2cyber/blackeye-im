# blackeye-im
This is a improved version of blackeye with ngrok support which was not working in the previous blackeye.

Installation:

git clone https://github.com/Git-Ankitraj/blackeye-im.git
cd blackeye-im
chmod +x ./blackeye.sh

Usage:

./blackeye.sh
